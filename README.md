# LPCGRA-Flow: A Design Methodology for LP/ULP CGRAs

**LPCGRA-Flow** provides an open-source toolchain and a methodological framework for designing Low-Power (LP) and Ultra-Low-Power (ULP) Coarse-Grained Reconfigurable Architectures (CGRA).

As edge computing and Internet of Things (IoT) devices demand increasingly efficient accelerators, CGRAs present a highly adaptable solution. However, standard generation tools often overlook aggressive power-saving techniques. **LPCGRA-Flow** addresses this gap by integrating ULP design principles directly into the architectural generation and exploration steps.

### What's Inside?
1. **Design Steps:** Comprehensive documentation detailing the stages of ULP CGRA design.
2. **Toolflow:** Scripts and utilities for parameterizing and generating power-optimized reconfigurable fabrics.
3. **Benchmarks & Templates:** Baseline architectures and examples to validate power metrics and functionality.
